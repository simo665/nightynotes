NightyNotes is a note-taking application developed as an educational exercise. The code and structure reflect learning practices and may not follow all production-ready standards.

## Getting Started

### Prerequisites
- Flutter SDK installed
- Dart SDK
- Firebase account (for authentication and storage features)
- IDE (VS Code, Android Studio, or IntelliJ)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/nightynotes.git
cd nightynotes
```

2. Install dependencies:
```bash
flutter pub get
```

3. Configure Firebase:
   - Add your Firebase configuration files (`google-services.json` for Android, `GoogleService-Info.plist` for iOS)
   - Update Firebase settings in the app

4. Run the app:
```bash
flutter run
```
